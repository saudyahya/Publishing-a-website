Uploading and publishing a website

The acronym ftp stands for:

File Transfer Protocol.

Here are 3 things needed to upload and publish the content of a website using ftp:

  • FTP client software.

  • All files in one folder.

  • Host name for Ftp.

  • Host port to be used for upload.

  • Username and password
